## Install 

- First, make sure you have Expo CLI installed: `npm install -g expo-cli`

- Install dependencies: `npm i` or `npm install`

- Run Project Locally: `npm start`

## Features

- User authentication
- Create a group chat
- Send messages

## Technologies

- React Native
- Firebase (Authentication and Firestore database)
- React Navigation
- Expo
